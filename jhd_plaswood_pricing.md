## 💰 สูตรคำนวณราคาขาย

### ขั้นตอนที่ 1 — ต้นทุนจริง (รวม VAT)
```
ต้นทุนจริง = ราคาทุน × 1.07
```

### ขั้นตอนที่ 2 — ราคาขายตามประเภทลูกค้า
```
ราคาขาย = ต้นทุนจริง × Multiplier
```

| ประเภทลูกค้า | Multiplier | หมายเหตุ |
|---|:---:|---|
| 🏪 ปลีก (Retail) | × 1.667 | ลูกค้าทั่วไป |
| 🔧 ช่างพาร์ทเนอร์ (Partner) | × 1.43 | ช่างที่ซื้อประจำ |
| 🤝 ดีลเลอร์ (Dealer) | × 1.25 | ตัวแทนจำหน่าย |
| 📦 ขายส่ง (Wholesale) | × 1.11 | ออเดอร์ปริมาณมาก |

### ขั้นตอนที่ 3 — ปัดขึ้นทุก 50 บาท
```
ราคาขายสุดท้าย = ปัดขึ้น (ราคาขาย / 50) × 50
```

> **ตัวอย่าง:** CT 10มม. ราคาทุน 1,883.20 บาท
> - ต้นทุนจริง = 1,883.20 × 1.07 = 2,015.02
> - ราคาปลีก = 2,015.02 × 1.667 = 3,359.02 → ปัดขึ้น = **3,400 บาท**
